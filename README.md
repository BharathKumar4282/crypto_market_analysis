# Crypto Market Intelligence Dashboard

An end-to-end data analytics project using real-time cryptocurrency data.

## Features
- Real-time data extraction from CoinGecko API
- Data cleaning & transformation using Python
- MySQL database integration
- Interactive Power BI dashboard
- Market cap, volume & volatility analysis

## Tech Stack
- Python (Pandas, Requests, SQLAlchemy)
- MySQL
- Power BI
- REST API
